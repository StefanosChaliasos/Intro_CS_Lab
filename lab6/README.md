
## sizeof
* μέγεθος πίνακα: sizeof(a)
* μέγεθος στοιχείου: sizeof(a[0])
* μήκος πίνακα: sizeof(a) / sizeof(a[0])
* cast: (int) (sizeof(a) / sizeof(a[0]))
* define: #define SIZE ((int) (sizeof(a) / sizeof(a[0])))
